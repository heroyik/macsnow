# Performance Log

## Objective

Track repeatable Activity Monitor observations while tuning the main implementation snow layers.

## Measurement Setup

- Build: `swift build`
- Launch: `swift run Xsnow`
- Tool: Activity Monitor
- View: CPU and Memory tabs
- Test duration: at least 3 minutes per scenario
- Recommended windows: Finder, Safari, and one developer app visible while accumulation is enabled

## Scenarios

| Scenario | Settings | Expected Result | CPU | Memory | Status | Notes |
| --- | --- | --- | --- | --- | --- | --- |
| Idle menu app | Snow stopped | No overlay animation or scanner activity. | Pending | Pending | Pending |  |
| Normal snow | Density Normal, Physics Balanced, Accumulation On | Smooth snow and bounded physics flakes. | Pending | Pending | Pending |  |
| High visual load | Density High, Physics High, Detailed accumulation | App remains usable without obvious stutter. | Pending | Pending | Pending |  |
| Spill stress | Density High, Physics High, Detailed accumulation, Spill When Full On | Detailed spill fragments remain bounded and clear after animation. | Pending | Pending | Pending |  |
| Fullscreen friendly | Overlay Level Fullscreen Friendly | Overlay remains click-through and less intrusive over fullscreen Spaces. | Pending | Pending | Pending |  |
| Display change | Connect or disconnect a display | Overlays rebuild without runaway CPU or memory growth. | Pending | Pending | Pending |  |
| Sleep/wake | Sleep and wake the Mac | Rendering and scanner stop during sleep and resume according to settings. | Pending | Pending | Pending |  |

## Tuning Notes

- Physics Quality Low: lowest contact-driven load for older Macs or many displays.
- Physics Quality Balanced: current default target for normal use.
- Physics Quality High: visual stress setting; use for tuning only if Activity Monitor remains acceptable.
- Accumulation Style Detailed adds more shape nodes than Soft or Layered.
- Overlay Level Fullscreen Friendly uses a lower `.floating` level than Normal to reduce fullscreen and Stage Manager interference.
